# EBA Chat Sistemi

Bu proje, EBA benzeri bir eğitim platformuna **bireysel ve grup tabanlı chat sistemi** eklemeyi amaçlar. Öğrenciler, öğretmenler ve idareciler hem birebir hem de grup sohbetleri gerçekleştirebilecekler.

## Kullanılan Teknolojiler
- HTML / CSS / JavaScript (Arayüz)
- Node.js + Express (Backend planı)
- SQL Server (Veritabanı planı)
- JWT Authentication (Kimlik doğrulama planı)
- bcrypt (Şifreleme planı)
- Socket.io (Gerçek zamanlı iletişim planı)
